# DSA Mind Map - Pakistan Edition 🇵🇰

An interactive visual mind map for mastering Data Structures and Algorithms with a focus on interview preparation and real-world applications.

## 🎯 Overview

DSA Mind Map is a comprehensive, interactive visualization tool designed to help you understand and master the most important algorithms and data structures. Built specifically with Pakistani developers in mind, this tool presents complex DSA concepts in an intuitive, interconnected mind map format.

## ✨ Features

- **Interactive Mind Map Visualization**: Explore algorithms and data structures in a beautiful, zoomable, interactive graph
- **Search Functionality**: Quickly find algorithms and topics with the integrated search feature
- **Priority-Based Learning**: Algorithms are categorized by importance:
  - **🔴 MUST KNOW**: Essential for every interview
  - **🟢 HIGH**: Very commonly asked
  - **🔵 GOOD**: Nice to have knowledge
- **Detailed Algorithm Cards**: Click any algorithm to see:
  - Time and space complexity
  - Real-world situations where it applies
  - When and how to use it
  - Interview recommendations
  - Code implementation examples
- **Statistics Dashboard**: Track progress with:
  - Count of Must Know, High, and Good algorithms
  - Total topics and algorithms covered
- **Smooth Interactions**: Drag to pan, scroll to zoom, click to explore
- **Beautiful Dark Boxy UI**: Modern, professional interface with dark theme elements

## 🚀 Quick Start

1. Open `index.html` in your web browser
2. **Search**: Use the search bar to find specific algorithms
3. **Explore**: Click on any topic (central node) to expand branches
4. **Drill Down**: Click on algorithms to see detailed information in the side panel
5. **Navigate**: 
   - Scroll to zoom in/out
   - Drag to pan around the map
   - Use the `+`, `−`, and `⊡` buttons for zoom and fit-to-view

## 📊 Covered Topics

The mind map covers fundamental data structure and algorithm topics:

- **Arrays**: Binary Search, Two Pointers, Sliding Window, Prefix Sum
- **Strings**: Pattern Matching, Subsequences, Substrings
- **Trees**: Traversals, BST, Balancing
- **Graphs**: DFS, BFS, Shortest Paths
- **Dynamic Programming**: Optimization, State Transitions
- **Sorting & Searching**: Classic algorithms with optimizations
- **And many more...**

## 🎓 How to Use for Interview Prep

1. **Identify Priority**: Focus on algorithms marked as "MUST KNOW" first
2. **Understand Patterns**: Notice how different algorithms relate to each other
3. **Practice Implementation**: Use the provided code samples as a base
4. **Study When & Why**: Read the real-world situations and recommendations
5. **Explore Variations**: Understand different approaches to similar problems

## 💡 Algorithm Information Breakdown
